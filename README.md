# Python SQL Framework Library (PySFL)

___
### Структура проекта
- База данных
    - Подключение к базе данных
    - Управление
        - Создание таблиц
        - Удаление таблиц
        - Сохранение базы данных
- Таблицы
    - Создание таблицы
        - Получение полей
    - Изменение таблицы
        - Изменение полей
        - Изменение названия
    - Упраление объектами таблицы
        - Получение объектов
        - Добавление объекта
        - Изменение объектов
        - Удаление объектов
